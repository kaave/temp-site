about TWWARR

Tokai Web Worker's Association for Rare Restaurants.
希少飲食店のための東海Web系業務者組合

BIOGRAPHY

2016年12月、会員の転職に伴う新たな歩みの幸運を祈って発足。
以後、おおよそ2,3ヶ月毎に開催。

メンバーをいたずらに増やさず、1台のテーブル、1枚の皿を全員で共有し、
Webの現状・未来、ときどき想い出話を肴にあまり見かけない各国料理のレストランを食べ歩くことを存在意義としている。
